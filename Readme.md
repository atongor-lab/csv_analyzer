Приложение для анализа CSV файлов (Streamlit)

О проекте
Проект представляет собой разработку приложения "Анализатор CSV файлов" с помощью streamlit

Описание:
Анализатор CSV файлов предназначен для быстрого исследования табличных данных, сохраненных в формате таблицы с разделителем. Программа автоматически определяет кодировку и разделитель с возможностью их переопределения, проводит базовый статистический и корреляционный анализ, предоставляет возможности построения некоторых графиков зависимости двух величин и группировки с агрегирующими функциями. Обрабатывает даты для более правильного построения интерактивных графиков.

Основные возможности:
- Загрузка CSV – автоматическое распознавание кодировки (библиотека chardet) и разделителя csv.Sniffer с возможностью переопределения параметров в ручную.
- Просмотр и редактирование – отображение данных в интерактивной таблице st.dataframe с возможностью изменения отключения не информативных столбцов.
- Статистический анализ – вычисление среднего, медианы, стандартного отклонения, минимума, максимума и квантилей для выбранных числовых столбцов.
- Визуализация – построение графиков с помощью Plotly:
  - Линейный график (только числовые признаки).
  - Диаграмма рассеяния (только числовые признаки).
  - Столбчатая диаграмма.
  - Гистограмма распределения.
  Ко всем графикам можно применить группировку по стоблцу Х и агрегацию по столбцу Y (сумма, среднее, количество)
  Отдельно вынесен корреляционный анализ числовых показателей с использованием тепловой карты
- Экспорт графиков – скачивание любого графика в формате PNG.
- Помощь – Описание работы некоторых элементов программы.

В приложении используются методы кэширования Streamlit @st.cache_data и st.session_state.

Установка зависимостей:
Для запуска программы необходим Python 3.10 или выше.
Необходимые пакеты

pip install streamlit pandas chardet plotly kaleido

Запуск:
Перейдите в папку с проектом и выполните команду

streamlit run .\app.py

Браузер открывается автоматически по адресу http://localhost:8501 (порт может меняться, актуальный порт указывается в консоли). Откройте любой CSV-файл (расширения .csv) и начните анализ.
